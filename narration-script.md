# 美妆成分兼容性讲解脚本

## 段落1: 开场引入
- **讲解文案**: 闺蜜们！你有没有过这种经历——买了一堆精华，涂完发现脸红了？今天我用一张图，帮你搞懂护肤成分到底能不能一起涂！
- **动画指令**: 画面淡入，展示标题"成分能不能一起涂？"，手绘风格边框逐步画出
- **高亮元素**: 标题文字、手绘边框

## 段落2: 认识成分家族
- **讲解文案**: 先认识一下我们桌上常见的12位主角。美白组的VC、烟酰胺、传明酸；酸类的果酸和水杨酸；抗老三巨头A醇、胜肽、玻色因；还有修复担当神经酰胺、保湿万能的玻尿酸，别忘了防晒！
- **动画指令**: 逐个弹出成分框，每框包含成分名和功效标签，按类别分组出现
- **高亮元素**: 每个新出现的成分框（依次高亮后变暗）

## 段落3: 神仙CP们
- **讲解文案**: 好消息先说！有些成分在一起简直是1加1大于2。烟酰胺加A醇，这个组合被吹爆了——A醇抗老，烟酰胺还能帮它降低刺激。玻色因加胜肽，一个从底层撑起皮肤，一个从结构加固，抗老黄金搭档！
- **动画指令**: 画出绿色兼容连线（烟酰胺↔A醇、玻色因↔胜肽、胜肽↔神经酰胺），连线带脉冲动画，两端成分框发光
- **高亮元素**: 绿色连线、被连接的成分框

## 段落4: 万能百搭选手
- **讲解文案**: 还有个好消息——玻尿酸是万能百搭选手！它就是个基础保湿成分，跟谁都不打架，放心叠。神经酰胺搭配A醇也是绝配，帮你把A醇的刺激拉下来一个档次。
- **动画指令**: 从玻尿酸画出多条绿色细线连接到各个成分框；再画出神经酰胺↔A醇的绿色线
- **高亮元素**: 玻尿酸（中心位置发光）、神经酰胺↔A醇连线

## 段落5: 需要间隔的组合
- **讲解文案**: 接下来是黄牌警告区！VC和烟酰胺，网上说法很多——其实现代研究表明正常浓度下问题不大，但敏感肌还是建议早晚分开。A醇和VC也别一起用，经典的"早C晚A"就是让它们各上各的班。
- **动画指令**: 画出黄色虚线（VC↔烟酰胺、A醇↔VC、烟酰胺↔酸类），每条线出现时两端成分框闪黄
- **高亮元素**: 黄色虚线、⚠️标记

## 段落6: 绝对不要一起的
- **讲解文案**: 红牌来了！果酸加水杨酸，等于双重去角质，屏障直接裂开。A醇加果酸或水杨酸也是一样的道理——两个刺激成分叠加，那是作死不是护肤！新手一定要记住这三对冤家。
- **动画指令**: 画出红色虚线（果酸↔水杨酸、A醇↔果酸、A醇↔水杨酸），带❌标记，两端成分框闪红
- **高亮元素**: 红色虚线、❌标记

## 段落7: 实用搭配方案
- **讲解文案**: 最后给你们一个实操方案！经典早C晚A：早上VC衍生物加防晒，晚上A醇加神经酰胺面霜。想美白就传明酸加烟酰胺，想温和抗老就胜肽加玻色因。记住一个原则——刷酸的时候别加A醇，敏感肌宁可不叠也不要叠！
- **动画指令**: 画面底部依次浮现方案卡片（早C晚A、美白攻坚、温和抗老、刷酸修复），每张卡片带手绘边框
- **高亮元素**: 方案卡片
